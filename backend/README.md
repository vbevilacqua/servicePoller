= Backend of Service Poller

== How to run locally

Use the following command:
```
./gradlew clean run
```

== How to test

Use the following command:
```
./gradlew clean test
```

## Technologies
    - Java 11
    - Spring Boot
    - Spring JPA
    - Lombok
    - H2
